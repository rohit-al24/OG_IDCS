About IDCS Portal

Overview

IDCS is a comprehensive academic and student services portal designed for K.Ramakrishnan College of Technology. The platform provides role-based interfaces for students, staff, HOD, and AHOD, along with public pages and administrative functions.

Key Features

  • Student Portal: Access dashboard, attendance records, timetables, and apply for bonafide certificates, gate passes, leave requests, and on-duty forms
  • Staff Portal: Manage class schedules, handle student requests, send notifications, and track academic activities
  • HOD/AHOD Portal: Review and approve applications, view mentee lists, and access department-specific dashboards
  • ATS System: Advanced resume analysis and tracking system for placement activities
  • PathPilot: Career guidance and academic pathway planning tools
  • Resume Builder: Professional resume creation tools for students
  • Feed360: Comprehensive feedback system for continuous improvement

Technologies Used

  • Built with Django framework for robust backend operations
  • Modern responsive design for seamless mobile and desktop experience
  • Secure authentication and role-based access control
  • Real-time notifications and updates
